## Method names

Methods are named based on type of solution: iterate (for loops), recursive, pattern_match, std (standard api)

## Running tests

### Golang

`go test ./... -v`

### Ruby

`ruby **/test_*.rb -v`

### Elixir

`elixir **/*_test.exs`

### Typescript & Deno

`deno test **/*_test.ts`
