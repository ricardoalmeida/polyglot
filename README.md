Algorithms in A, B and C just for fun! _へ__(‾◡◝ )>

## Method names

Methods are named based on type of solution: iterate (for loops), recursive, pattern_match, std (standard api)

## Running tests

### All tests

`rake -m test`. Use -v for verbose mode.

### Golang

`go test ./... -v`

### Ruby

`ruby **/test_*.rb -v`

### Elixir

`mix test`

### Deno & Typescript

`deno test **/*_test.ts`

### Rust

`cargo test`
